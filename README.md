# IMDb Movie Rating Scraper

## Project Description

The IMDb Movie Rating Scraper is a Python-based automation tool that uses Selenium and Chrome WebDriver to retrieve movie details from IMDb.

## Features

- Dynamic Movie Scraping
- Top Movie Rankings
- CSV File Export
- Selenium Browser Automation
- Easy to Extend

## Technologies Used

- Python
- Selenium
- Pandas
- WebDriver Manager

## Output

The project generates a CSV file containing:

- Rank
- Movie Title
- Release Year
- IMDb Rating

## Run the Project

```bash
python imdb_scraper.py
```
